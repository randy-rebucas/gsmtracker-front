<h1>Introduction</h1>

Myclinicsoft is a Web Application for all Medical Practitioners. 
The main modules :
1. Patients
2. Multiuser with permission control
3. Role Base Permissions
4. Templating
5. Dynamic Records
6. Reporting
7. Appointments
8. Queings

Supported functions :
1. RX printing and/or emailing
2. Multiuser with permission control
3. Messaging (SMS)
4. Multilanguage
5. Custom Fields

Supported Patient Records :
1. Temperature
2. Weight
3. Endorsement
4. Next visit
5. Medications
6. Height
7. Family History
8. Conditions
9. Blood Pressure
10. Blood Glucose
11. Allergies
12. Immunization
13. Lab test Results
...Also can support dynamic type of records

The software is written in PHP language, it uses MySQL (or MariaDB) as data storage back-end and has a simple but intuitive user interface.
